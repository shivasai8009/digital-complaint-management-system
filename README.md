# digital-complaint-management-system
A web-based Digital Complaint Management System built using Java, JSP, JDBC, MySQL, and Bootstrap. Allows Customers to raise complaints, Solvers to resolve them, and Admins to manage the entire workflow with real-time tracking, notifications, and role-based access.
